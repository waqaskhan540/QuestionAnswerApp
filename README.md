# QuestionAnswerApp
A simple app for asking questions and posting answers, built using React and ASP.NET Core

# Requirements

- .NET Core SDK 2.2
- MySQL
- MySQL Connector

# How to Setup ?

1. `cd` into the `api` project
    >`cd server\Server\Api`

2. restore nuget packages
    >`dotnet restore`

3. build the project
    >`dotnet build`

4. run the project
    >`dotnet run`
